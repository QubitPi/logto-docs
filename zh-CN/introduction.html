简介

欢迎来到 Logto 文档！Logto 是为现代应用和 SaaS 产品设计的身份与访问管理 (IAM) 解决方案。它为你的应用程序提供安全、可扩展且可定制的认证 (Authentication) 和授权 (Authorization) 系统。

按功能探索​

注册和登录企业单点登录 (SSO)多因素认证 (MFA)认证 (Authentication) 参数魔法链接（一次性令牌）用户授权页面 (Consent screen)登出账户设置安全验证组织 (Organization) 体验

密码策略CAPTCHA 机器人保护黑名单标识符锁定

基于角色的访问控制 (RBAC) (Role-based access control)组织模板

用户管理组织 (Organizations)

自定义令牌声明 (Custom token claims)用户模拟 (User impersonation)Webhook审计日志

按支持工具包探索​

Logto 的能力建立在四大支柱之上，构成了你可以轻松集成到产品或业务中的功能基础。

基于 Web 的界面，用于配置和管理资源，快速搭建登录体验并轻松进行身份管理。

提供完整的认证 (Authentication) 流程和用户界面，并可通过 Logto Console 和 API 进行广泛定制。

Logto 后端提供一套 API，支持多种认证 (Authentication) 和授权 (Authorization) 功能，包括 Logto Management API、Experience API 和 Account API。

探索我们的端到端教程和 SDK，快速上手你喜欢的开发框架。

按部署类型探索​

我们的云服务包含 Free、Pro 和 Enterprise 计划，满足不同客户需求。由 Logto 托管和管理，同时提供 dev tenant，包含 Logto 的全部功能用于测试。

在你自己的基础设施上自托管 Logto。访问我们的 GitHub 页面和开源专区了解更多详情。

加入 Logto Discord 服务器 —— 这里是开发者和企业的活跃社区。参与讨论、获取最新动态并分享你的反馈。
