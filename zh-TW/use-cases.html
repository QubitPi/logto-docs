使用情境

Logto 是一套強大的驗證 (Authentication) 基礎設施與身分與存取管理（IAM, Identity and Access Management）解決方案，能應用於各種場景。我們整理了一系列使用情境，協助你瞭解如何在不同情境下善用 Logto。

授權 (Authorization)​

角色型存取控制 (RBAC, Role-based Access Control) 完整指南：精通權限設計、角色管理，並透過實際 CMS 實作強化安全授權 (Authorization)。

多租戶 (Multi-tenancy)​

學習如何以高效方式打造具備強大驗證 (Authentication)、組織管理 (Organization management) 與角色型存取控制 (RBAC, Role-based access control) 的多租戶 SaaS 應用程式。

AI​

探索 Logto 如何協助你打造安全、支援 AI 的使用體驗 (Experience)。無論你是在實作 Model Context Protocol (MCP)，還是將 AI agent 整合進你的應用程式，Logto 都提供開發者友善的驗證 (Authentication) 與授權 (Authorization) 解決方案。

打造你的 MCP 伺服器

學習如何將驗證 (Authentication) 與授權 (Authorization) 整合進你的 MCP 伺服器，讓你的應用程式與服務能安全存取。

了解如何為連接到你 MCP 伺服器的第三方 AI agent 與應用程式啟用驗證 (Authentication) 與授權 (Authorization)。

為你的應用程式打造 AI-ready 驗證 (Authentication)

探索如何為第三方 AI agent 存取你的應用程式新增驗證 (Authentication) 與授權 (Authorization)，確保 AI 驅動流程的安全與控管。
