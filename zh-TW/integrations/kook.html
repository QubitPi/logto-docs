設定 KOOK 社交登入 (Set up social login with KOOK)

官方 Logto KOOK 社交登入 Web 應用程式連接器。

提示:

本指南假設你已對 Logto 連接器 (Connectors) 有基本了解。若不熟悉，請參閱 連接器 (Connectors) 指南以開始使用。

開始使用​

KOOK 連接器讓終端使用者可以透過 KOOK OAuth 2.0 驗證協議，使用自己的 KOOK 帳號登入你的應用程式。

在 KOOK 開發者中心建立應用程式​

提示：如果你已經完成部分步驟，可以略過相關章節。

備註:

你需要先申請 KOOK OAuth2 權限，才能建立 OAuth2 應用程式。請參閱 OAuth2 頁面資訊，或在 KOOK 官方開發者伺服器諮詢 KOOK 工作人員以獲得更多資訊。

  1. 前往 KOOK 開發者中心 並使用你的 KOOK 帳號登入。如果還沒有帳號，可以註冊新帳號。
  2. 點擊 建立應用程式 (Create Application) 按鈕來建立應用程式，為其命名（例如 LogtoAuth），然後點擊 確認 (Confirm)。
  3. 點擊你剛建立的應用程式圖示，進入 OAuth2 頁面。
  4. 在 Callback URIs 欄位新增有效的重導 URI（例如 ${your_logto_origin}/callback/${connector_id}）。你可以在 Logto 管理主控台的連接器詳細頁面找到 redirect uri。
  5. 別忘了在 OAuth2 連結產生器 (OAuth2 link builder) 勾選 get_user_info 權限範圍 (scope)。

設定你的 KOOK 連接器 (Connector)​

將你在上一節 OAuth2 頁面取得的 Client ID 與 Client Secret 分別填入 clientId 與 clientSecret 欄位。

設定類型​

NameType
clientIdstring
clientSecretstring
scopestring

測試 KOOK 連接器 (Connector)​

就是這麼簡單。KOOK 連接器現在應該可以使用了。別忘了 在登入體驗 (Sign-in experience) 啟用連接器。
