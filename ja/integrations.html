はじめに

Logto コネクターのドキュメントへようこそ。このドキュメントには、Logto コネクターを介してサードパーティの認証サービスを Logto と統合するための有用なガイドが含まれています。

ソーシャルコネクター

メールコネクター

Aliyun Direct MailAliyun はオンラインビジネス向けにクラウドコンピューティングサービスを提供しています。AWS ダイレクトメールAmazon SES は、バルクメール送信のために任意のアプリケーションに統合できるクラウドメールサービスプロバイダーです。HTTP EmailHTTP コールでメールを送信します。MailgunMailgun は、メールの送信、受信、および追跡のためのメール配信サービスです。PostmarkPostmark はメール送信プラットフォームです。SendGrid EmailSendGrid は、トランザクションおよびマーケティングメールのためのコミュニケーションプラットフォームです。SMTPSMTP は電子メール送信のためのインターネット標準通信プロトコルです。

SMS コネクター

Aliyun Short MessageAliyun はオンラインビジネス向けにクラウドコンピューティングサービスを提供しています。GatewayAPI SMSGatewayAPI は、学習曲線と推測作業を排除することで開発を加速し、すぐに API を使用して構築を開始できるようにします。SMS AeroSMS Aero は、契約を確認することなく 5 分で SMS メーリングを利用できるようにします。開発者には、アクセス可能なクラスと 24x7 チャットサポートを備えた便利な API が提供されます。Tencent Short MessageTencent はオンラインビジネス向けにクラウドコンピューティングサービスを提供しています。Twilio SMSTwilio は電話やメッセージのためのプログラム可能な通信ツールを提供します。Vonage SMS世界をつなぐコミュニケーション API。
