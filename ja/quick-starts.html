クイックスタート

Logto クイックスタートへようこそ。このドキュメントには、アプリケーションへ Logto を統合するための有用なガイドが含まれています。

ほとんどのガイドは Logto 公式 SDK を使用しています。ご不明な点があれば、どうぞお気軽にお問い合わせください。

備考:

このページは Logto クライアント統合（バックエンド・フォー・フロントエンドパターンを含む）に焦点を当てています。サーバーサイド API 保護については API 保護 をご覧ください。

言語フレームワーク

Android (Kotlin / Java)Android 統合ガイド。AngularAngular はユーザーインターフェースを構築するための JavaScript ライブラリです。Capacitor JSCapacitor JS はハイブリッドモバイルアプリを構築するためのフレームワークです。Chrome extensionChrome 拡張機能は、Google Chrome 内で実行されるミニ Web アプリケーションです。.NET Core (Blazor Server)Logto を .NET Core Blazor Server アプリに統合する。.NET Core (Blazor WASM)あなたの .NET Core Blazor WebAssembly アプリに Logto を統合します。.NET Core (MVC)Model-View-Controller (MVC) を使用して .NET Core Web アプリに Logto を統合します。.NET Core (Razor Pages)Razor Pages とビューモデルを使用して .NET Core Web アプリに Logto を統合する。Expo (React Native)Android、iOS、および Web で動作する React を使用してユニバーサルネイティブアプリを作成します。ExpressExpress は、ウェブおよびモバイルアプリケーションのための堅牢な機能セットを提供する、最小限で柔軟な Node.js ウェブアプリケーションフレームワークです。FlutterFlutter はハイブリッドモバイルアプリを構築するためのフレームワークです。FlutterFlowFlutterFlow はハイブリッドフラッターアプリを構築するためのローコードフレームワークです。GoGo は、Web アプリケーション向けの効率的で並行性のある静的型付けプログラミング言語です。Java Spring BootSpring Boot は、Java プログラミング言語を使用して、安全で高速かつスケーラブルなサーバーアプリケーションを構築するための Java 用 Web フレームワークです。Next.js (Pages Router)Next.js はプロダクション向けの React フレームワークで、フルスタックの React アプリを簡単に構築でき、組み込みの SSR を備えています。Next.js (App Router)Next.js App Router は、React の最新機能を使用してアプリケーションを構築するための新しいパラダイムです。Auth.js (Next Auth)Next.js のための認証 (Authentication)。NuxtNuxt は、ウェブ開発を直感的かつ強力にするオープンソースのフレームワークです。Passport.jsPassport is authentication middleware for Node.js.PHPLogto を PHP Web アプリケーション（Laravel など）に統合する。PythonDjango や Flask などの Python ウェブアプリに Logto を統合します。ReactReact はユーザーインターフェースを構築するための JavaScript ライブラリです。RubyRuby は、シンプルさと生産性に重点を置いた動的なオープンソースのプログラミング言語です。SvelteKitSvelteKit は、Svelte を使用して堅牢で高性能な Web アプリケーションを迅速に開発するためのフレームワークです。iOS (Swift)iOS (Swift) アプリケーション統合ガイド。Vanilla JSフレームワークに依存しない JavaScript 統合。VueVue.js は、ウェブ上で UI を構築するための進歩的で段階的に採用可能な JavaScript フレームワークです。WebflowWebflow は、ウェブサイトの構築とホスティングのための SaaS プラットフォームです。WordPressWordPress は無料でオープンソースのコンテンツ管理システムです。WordPress プラグイン公式の WordPress プラグインを使用して、WordPress ウェブサイトに Logto を統合します。

フレームワークなしおよび一般

マシン間通信マシン間での直接通信を可能にします。Traditional webサーバー上で動作する従来の Web アプリに Logto を統合する。

データベース

HasuraLogto と Hasura を統合する。SupabaseLogto を Supabase と統合する。

サードパーティアプリ (Logto as Idp)

OIDC（サードパーティアプリ）アプリケーションのために Logto をサードパーティ OIDC アイデンティティプロバイダー (IdP) として利用する。
