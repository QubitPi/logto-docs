import createApplication from '@site/docs/integrations/sso/entra-id-oidc/assets/entra_id_oidc_create_application.webp';

1. 前往 [Microsoft Entra 管理中心](https://entra.microsoft.com/) 並以管理員身分登入。

2. 瀏覽至 Identity > Applications > App registrations。

<center>
  <img src={createApplication} alt="Create Application" />
</center>

3. 選擇 `New registration`。

4. 輸入應用程式名稱，並選擇適合你的應用程式的帳戶類型。

5. 選擇 `Web` 作為應用程式平台。

6. 從 Logto 的 SSO 設定頁面複製並貼上 `redirect URI`。`redirect URI` 是使用者在 Microsoft Entra ID 驗證後被重定向的 URL。

<center>
  <img src={configApplication} alt="Configure Application" />
</center>

6. 點擊 `Register` 以建立應用程式。
