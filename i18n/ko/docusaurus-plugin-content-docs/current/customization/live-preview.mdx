# 실시간 미리보기

**실시간 미리보기** 기능을 사용하여 로그인 경험 구성의 편리한 테스트를 할 수 있습니다.

<CloudLink to="/sign-in-experience">콘솔 > 로그인 경험</CloudLink>으로 이동하여 오른쪽 상단의
"실시간 미리보기" 버튼을 클릭하면 Logto의 기능을 완전히 시연하는 새로운 브라우저 탭이 열립니다. 이
데모는 로그인 및 회원가입 흐름에 대한 우리의 경험 디자인 접근 방식을 보여줍니다. 매우 커스터마이즈
가능하며 애플리케이션에 원활하게 통합될 수 있습니다.

:::note

**참고**: 이 미리보기에서 경험을 테스트하려면 먼저 계정을 생성해야 합니다. 이 계정은 나중에 <CloudLink to="/user-management">콘솔 > 사용자 관리</CloudLink>에서 관리할 수 있습니다.

:::

생성한 사용자 계정으로 로그인 시도를 하고 문제가 없이 성공하면, 식별자와 ID가 표시된 성공 페이지로 리디렉션됩니다.
