빠른 로그인과 사용자 전환율을 개선하기 위해 {props.framework}를 아이덴티티 제공자 (IdP)로 연결하세요. Logto 소셜 커넥터는 여러 매개변수 입력을 허용하여 몇 분 안에 이 연결을 설정할 수 있도록 도와줍니다.

소셜 커넥터를 추가하려면 다음 단계를 따르세요:

1. [Console > Connectors > Social Connectors](https://cloud.logto.io/to/connectors/social)로 이동합니다.
2. "소셜 커넥터 추가"를 클릭하고 "{props.connector}"를 선택합니다.
3. README 가이드를 따라 필수 필드를 완료하고 설정을 사용자 정의합니다.

<img src={connectorTabSocialImage} alt="Connector tab" />

:::
