{props.connectorType} 커넥터는 인증을 위해 일회용 비밀번호 (OTP)를 전송하는 데 사용되는 방법입니다. 이는 {props.passwordlessSignUpIdentifier} 인증을 가능하게 하여 {props.connectorType} 기반 등록, 로그인, 이중 인증 (2FA), 계정 복구를 포함한 비밀번호 없는 인증을 지원합니다. {props.connector}를 {props.connectorType} 제공자로 쉽게 연결할 수 있습니다. Logto {props.connectorType} 커넥터를 사용하면 몇 분 안에 이를 설정할 수 있습니다.

{props.connectorType} 커넥터를 추가하려면 다음 단계를 따르세요:

1. [Console > Connector > Email and SMS connectors](https://cloud.logto.io/to/connectors/passwordless)로 이동합니다.
2. 새 {props.connectorType} 커넥터를 추가하려면 "Set up" 버튼을 클릭하고 "{props.connector}"를 선택합니다.
3. 선택한 제공자의 README 문서를 검토합니다.
4. "Parameter Configuration" 섹션에서 구성 필드를 완료합니다.
5. JSON 편집기를 사용하여 {props.connectorType} 템플릿을 사용자 정의합니다.
6. {props.passwordlessSignUpIdentifier}로 인증 코드를 보내어 구성을 테스트합니다.
