Pour simplifier la gestion des accès et obtenir des garanties de niveau entreprise pour vos grands clients, connectez-vous avec {props.framework} en tant que fournisseur d'identité fédéré. Le connecteur SSO d’entreprise Logto vous aide à établir cette connexion en quelques minutes en permettant l'entrée de plusieurs paramètres.

Pour ajouter un connecteur SSO d’entreprise, suivez simplement ces étapes :

1. Accédez à [Logto console > Enterprise SSO](https://cloud.logto.io/to/enterprise-sso).

<img src={ssoPageImage} alt="Page SSO" />

2. Cliquez sur le bouton "Ajouter un connecteur d’entreprise" et choisissez votre type de fournisseur SSO. Choisissez parmi les connecteurs préconstruits pour Microsoft Entra ID (Azure AD), Google Workspace et Okta, ou créez une connexion SSO personnalisée en utilisant le protocole standard OpenID Connect (OIDC) ou SAML.
3. Fournissez un nom unique (par exemple, Connexion SSO pour Acme Company).

<img src={selectYourSsoProviderImage} alt="Sélectionnez votre fournisseur SSO" />

4. Configurez la connexion avec votre IdP dans l'onglet "Connexion". Consultez les guides ci-dessus pour chaque type de connecteur.

<img src={ssoConnectionImage} alt="Connexion SSO" />

5. Personnalisez l'expérience SSO et le **domaine de messagerie** de l’entreprise dans l'onglet "Expérience". Les utilisateurs se connectant avec le domaine de messagerie activé pour le SSO seront redirigés vers l'authentification SSO.

<img src={ssoExperienceImage} alt="Expérience SSO" />

6. Enregistrez les modifications.
